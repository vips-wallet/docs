# VIPSTARCOIN insight-api documents

## usage

### use CLI

```
npm install -g aglio --unsafe-perm
git clone https://github.com/vips-wallet/docs.git
cd docs/insight-api
aglio -i md/ja.md -s
```

### use Docker

```
git clone https://github.com/vips-wallet/docs.git
cd docs/insight-api
docker-compose up -d

# open http://localhost:3000/ in your browser
```
